About the website

The website as of right now (24-Apr-16) is straight HTML5 and CSS3. The blog will be a Jekyll based blog with a custom theme (designed by me). My old blog ran Jekyll and I loved it. Not having to use something like Wordpress is freeing as you don't have to worry about exploits and having to keep it (and dozens of plugins) up to date.

The fonts used on the website are: Pacifico for the Logo, Oswald for the headings, and Open Sans for the content. I use Font Awesome for the Twitter and Github logos in the footer as well.


Future plans for the site include:

  • Move CSS to SASS to clean it up.
  • Use Jquery UI to add some transitions.
  • Clean up the html to make everything look pretty.
  • Code more sites for the portfolio page, it's kind of barren at the moment.
  • Make the site more responsive.
